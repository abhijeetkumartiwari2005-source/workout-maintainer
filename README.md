# Workout Maintainer

Workout Maintainer is a web application designed to help you plan, track, and maintain your workout routines. You can create custom workout plans, track your progress, and even monitor your daily calorie intake. This application is built with React and Vite, and it uses local storage to persist your data.

## Features

-   **Create Custom Workouts:** Easily add new exercises to your daily workout plan.
-   **Track Your Progress:** Keep track of sets, reps, and weights for each exercise.
-   **Visualize Your Progress:** A checkbox-based system allows you to mark your completed sets for each exercise as you go.
-   **Calorie Tracker:** A dedicated module to help you monitor your daily calorie intake with a visual progress circle.
-   **Pre-made Plans:** View a gallery of pre-made workout plans to get started quickly.
-   **Persistent Data:** Your workout data and calorie information are saved in your browser's local storage, so your data is there when you come back.
-   **User-Friendly Interface:** A clean and simple interface built with Bootstrap.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need to have Node.js and npm (or yarn) installed on your machine.

-   [Node.js](https://nodejs.org/) (which includes npm)

### Installation & Running

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/workout-maintainer.git
    ```
2.  Navigate to the project directory
    ```sh
    cd workout-maintainer
    ```
3.  Install NPM packages
    ```sh
    npm install
    ```
4.  Run the development server
    ```sh
    npm run dev
    ```
5.  Open your browser and go to `http://localhost:5173` (or whatever port the console indicates).

## Usage

1.  **Plan Your Workout:** Click on "Plan today's workout" in the sidebar.
2.  **Add Exercises:** Type the name of an exercise in the input field and click "Add".
3.  **Finalize Your List:** Once you've added all your exercises, click "Finalize & Add Details".
4.  **Enter Details:** Fill in the sets, reps, and weight for each exercise.
5.  **Track Your Workout:** As you complete your sets, check the corresponding boxes to track your progress.
6.  **Calorie Tracker:** Navigate to the calorie tracker using the button on the home page. Set your daily goal and add your calorie intake throughout the day.
